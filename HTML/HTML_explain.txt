HTML(Hyper Text Markup Language):超文本标记语言

组成：标签、元素、结构（网页头和网页体）、属性

#常见HTML元素
开始标签        元素内容        结束标签        用法
<h1>            一级标题        </h1>       一级标题
<h2>            二级标题        </h2>       二级标题
<p>             段落文本        </p>        段落
<a>             描述链接的文本  </a>        超链接
<div>           其他元素或文本  </div>      块

#常见HTML属性与用法
属性       用法
class      为html元素定义一个或多个类名(classname)
id         定义元素的唯一id
href       用来定义链接
style      规定元素的行内样式(inline style)
